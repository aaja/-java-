# -java-
对锅游戏

玩法：
1.玩家数量不限制
2.牌数量不限制
3.玩家依次发牌，各发4张
4.出牌分两组，每组两张，即 头 ，尾
5.头是牌组中的小牌，尾是大牌

大小比较：
1.A~9代表自己点数，10及其以上要-10，即J代表1点，Q代表2点，K代表3点，大小王代表0点
2.单组牌中，两张牌点数相加，超过9点，就要-10。例如，手中是 k+A，代表3 + 1= 4点
3.比较单组牌的大小，点数大，则本回合胜出；
      如果点数一样，则比较手中红色牌（红桃，方块）的数量，红色牌数量多的，则大；
4.头跟尾都胜出的，表示玩家胜出；
  如果只有一组胜出，则平局；
  允许玩家手牌大小完全一样，则表示平；
